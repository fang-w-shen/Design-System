Introduction

Schema design system is built from the Foundation framework and therefore uses the grid system. Currently, Schema uses Foundation's XY Grid. It's the most modern grid system available however it does have some browser support issues.

In this section of the Design System we will only document how to use the grid system to acheive Schema's page layouts. To learn more about the grid please visit Foundation Grid Docs.
