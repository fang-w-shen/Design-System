--- title: Components ---

Component Overview

Schema components are a set of "containers" that come directly from the Zurb Foundation framework. Each component has specific uses for user experience and consistency reasons.

Please be sure to read how to use each component before placing it into an application. If you are unsure you can always contact a UX Designer for advice.

Component Docs

To avoid duplication of documentation Schema only documents how to display a component using HTML. For further documentation and instructions on a component, please visit the Foundation docs.
